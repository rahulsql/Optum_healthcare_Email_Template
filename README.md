# Optum Healthcare - Professional Email Template

This is a high-fidelity, mobile-responsive email template designed specifically for the healthcare sector, inspired by Optum's branding and design language. 

### 🚀 Project Overview
The goal of this project was to build a bulletproof email notification system that remains consistent across all major email clients, including legacy versions of Microsoft Outlook.

### 🛠️ Technical Features
- **Table-Based Layout:** Built using nested HTML tables to ensure 100% rendering consistency across Outlook 2007, 2010, 2013, and modern clients.
- **Mobile Responsive:** Utilizes CSS Media Queries to adapt the layout for smartphones (320px - 480px).
- **Typography & Branding:** Implemented Optum’s signature color palette (#E87722, #00587C) and clean typography for a professional healthcare feel.
- **Accessibility (A11y):** Included `role="presentation"` on layout tables to improve screen reader compatibility.
- **Dark Mode Optimization:** Designed with a clean background strategy to handle automatic color inversion in Gmail and Outlook.

### 💻 Tech Stack
* **HTML5:** Semantic email-safe markup.
* **CSS3:** Inline styles for maximum compatibility.
* **MSO Comments:** Specialized conditional code for Microsoft Office rendering fixes.

### 📂 How to View
1. Clone this repository.
2. Open `index.html` in any web browser.
3. Use Chrome DevTools (Inspect) to test responsiveness on different device sizes.
